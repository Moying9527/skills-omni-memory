# 压缩规则 (Compression Rules)

## 触发条件（满足任一即压缩）
- Token占用 > 预算70%
- 用户显式请求
- 新任务开始前
- 会话即将结束

## 三种压缩策略

| 策略 | 适用 | 方法 |
|------|------|------|
| **锚定迭代摘要** | 长对话历史 | 保留"决策→执行→结果"链，淘汰旧段 |
| **关键信息提取** | 技术讨论/决策 | 提取偏好/决策/约束/进度 |
| **结构化压缩** | 代码/配置/日志 | 按比例压缩（见SKILL.md） |

## 语义链完整规则（不可违）
- 保留"决策→执行→结果"逻辑链
- 保留用户明确偏好、约束、截止日期
- 最后5轮对话不压缩
- 截断检测：发现语义断裂→停止→回滚

## 锚点字段（锚定摘要输出格式）

```markdown
## Anchored State
- 决策: [时间]: [决策] → [结果]
- 变更: [时间]: [变更] → [影响]
- 错误: [时间]: [错误] → [修复]
- 下步: [优先级]: [待办]
```

## 快捷指令
| 指令 | 功能 |
|------|------|
| `/omni_compress` | 自动压缩 |
| `/omni_compress full` | 强制全量压缩 |
| `/omni_compress anchor` | 锚定摘要压缩 |
| `/omni_compress extract` | 提取关键信息 |
